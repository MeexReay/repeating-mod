# Repeating Mod (Fabric)

This mod can record your movements and play them back.

![Preview gif](preview.gif)

## Controls

Default Repeating Mod Buttons

```
Menu              |  J            
Toggle recording  |  not specified
Toggle replay     |  not specified
```
